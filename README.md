🦻 Zakadi - Intuitive Communication

Zakadi is an ultra-lightweight, single-page, installation-free web application designed to facilitate communication through real-time voice transcription.
Built primarily for accessibility (hearing-impaired individuals, noisy environments, language barriers), Zakadi transforms voice into text displayed in large, high-contrast characters.

✨ Key Features

🎙️ Real-time Transcription: Continuously listens and displays text as you speak, with a clear distinction between validated text and text currently being recognized.
🧠 Anti-Duplication Algorithm (Overlap Merge): Integrates smart mathematical logic to counter frequent word duplication bugs (especially on Chrome Android) during long dictations.
🎧 Active Noise Filter: Uses the Web Audio API to apply a hardware bandpass filter, helping to isolate the human voice in noisy environments.
🌗 Accessibility and High Contrast: Adaptive interface (Light/Dark Theme) based on system preferences, designed without frills for optimal readability.
🔍 Typography Adjustment: Dynamic text size control (zoom in/out) to adapt to each user's eyesight.
📱 Responsive Design: Mobile-optimized interface with a unique behavior in landscape mode (displays text as a smooth horizontal prompter/ticker).
⌨️ Hybrid Input: Built-in alternative keyboard input panel to take over from voice if needed.
🔗 Instant Offline Sharing: Native QR Code generator (no external API calls) to easily share the app with an interlocutor.
📋 Quick Export: Copy the entire transcription to the clipboard with a single click.

🎯 Use Cases

Accessibility (Deaf and Hard of Hearing): Use Zakadi as a pocket subtitler during face-to-face conversations. Hold out your phone for your interlocutor to speak, and read their words in large print.
Noisy Environments: In a bar, a factory, or on the street, activate the "Noise Filter" to better capture conversations and make yourself understood.
Presentations & Conferences: Project the app onto a screen (or use landscape mode) to provide live subtitles to your audience.
Note-taking & Dictation: Dictate your ideas on the fly and copy the finalized text to paste into your emails or documents.

🚀 How to use it?

Zakadi is a 100% "Client-side" application (zero server). You can access and run it instantly with no installation required.

🌐 Web Access (Recommended)
Simply click on the published web link to start using the app:
👉 Open Zakadi Live on GitHub Pages (or your custom GitHub Pages URL)
Note: Microphone access requires a secure connection (HTTPS), which is fully supported and automatically provided by GitHub Pages.

💻 Local Use
If you prefer to run it locally on your machine:
Download the index.html file from this repository.
Open it in any modern web browser (Google Chrome or Microsoft Edge are highly recommended for the best voice recognition compatibility).
Note: Some browsers restrict microphone access when running files locally (via file://). Using the hosted web version above is highly recommended.
